# Basic Hybrids project setup

Boilerplate for [Hybrids](https://hybrids.js.org)

## Commands

### Install

```
yarn
```

### Start development server

```
yarn start
```

### Build production version

```
yarn build
```

## What's included:

- Babel 7 & Webpack 2 configuration
- CSS import support
- Image import support
- Tests with jest
